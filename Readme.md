# EV Station Management System ⚡

A web-based system to manage EV charging stations, bookings, and users.

## Features
- User & Admin login
- Station management
- Booking system
- Role-based dashboard

## Tech Stack
- PHP
- MySQL
- HTML/CSS

## Setup
1. Import SQL file
2. Configure `database.php`
3. Run on XAMPP
